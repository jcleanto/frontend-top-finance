## Descrição

Microserviço de frontend com:
- O CRUD de Lançamentos Financeiros.

## Setup do projeto

```bash
$ npm install
```

## Compilar e rodar o projeto

```bash
# development
$ npm run dev

# script para rodar o module federation localmente sem erros
$ npm run start-mf
```

## Rodar os testes

```bash
# unit tests
$ npm run test
```

## Acessar aplicação

http://localhost:5002/finances
